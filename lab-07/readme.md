# Лабораторная работа №7

## Задачи

1. **Загрузка и сохранение модели многогранника из файла**
   - Формат файла: Wavefront OBJ
   - Отображение загруженной модели
   - Применение аффинных преобразований к загруженной модели

2. **Построение фигуры вращения**
   - Фигура вращения задаётся образующей (набор точек), осью вращения и количеством разбиений
   - Угол вращения: 360° / количество разбиений
   - Возможность задать образующую и построить фигуру вращения относительно выбранной координатной оси
   - Сохранение полученной модели в файл

3. **Построение графика двух переменных**
   - Сегмент поверхности задаётся функцией f(x, y) = z
   - Диапазоны отсечения [x0, x1], [y0, y1]
   - Количество разбиений по осям (шаг)
   - Отображение модели и применение аффинных преобразований
   - Сохранение модели в файл

## Run

1. установите необходимые зависимости

```bash
sudo apt install build-essential libsfml-dev
```

2. компиляция

```bash
g++ main.cpp -o main -lsfml-graphics -lsfml-window -lsfml-system
```

3. запуск

```bash
./main
```
